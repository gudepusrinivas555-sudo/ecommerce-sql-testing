# ecommerce-sql-testing
# 🛒 E-Commerce SQL Testing Project

## 📌 Description
This project demonstrates SQL skills for database testing in an E-commerce system. It includes database creation, data insertion, and validation queries used in real QA scenarios.

## 🧰 Tech Used
- MySQL
- SQL Queries
- Database Testing

## 📂 Project Files
- schema.sql → Database & tables creation
- data.sql → Sample test data
- queries.sql → Business queries
- test_cases.sql → QA validation queries

## 🧪 Test Scenarios Covered
- Data validation (users, products, orders)
- Join operations
- Aggregation (SUM, COUNT)
- Duplicate checks
- Foreign key validation

## 🚀 How to Run
1. Open MySQL / Workbench
2. Run `schema.sql`
3. Run `data.sql`
4. Run `queries.sql`
5. Run `test_cases.sql`
